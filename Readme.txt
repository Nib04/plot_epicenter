
PythonにてPlotlyというライブラリを用い震源分布を三次元でプロットしました。

閲覧する際は[Clone or download]にてダウンロードし、ローカル環境にて閲覧ください。
閲覧するにあたって特別する作業はなく、webブラウザがあれば閲覧可能です。


「epi_usgs.html」がUSGS(アメリカ地質調査所)のデータを用いた図、
使用データの条件は
・2018年内の地震
・範囲は日本の東西南北端点である北緯20~46度、東経123~154度
・地震の規模や震度の条件は特になし。
・データ数:1136個


「epi_jma.html」がJAM(気象庁)のデータを用いた図です。
使用データの条件は
・2018年内の地震
・震度4以上
・データ数:74個
